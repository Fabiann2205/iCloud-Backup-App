# Upload your Home Assistant Backups to iCloud

To install this app in Home Assistant:
1. Add the repository `https://github.com/Fabiann2205/iCloud-Backup-App` to your Home Assistant app repositories.
2. In Home Assistant, go to **Settings → Apps/Addons → Install App →** (three dots in the upper right corner) **→ Repositories**, then click **Add**.
3. After adding the repository, you will see this app in the app/addon store.
